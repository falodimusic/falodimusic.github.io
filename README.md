Paquete minimal listo para subir

Archivos incluidos:
- index.html
- styles.css
- script.js
- red-mark.svg (favicon / icono)
- FALODI LOGO BLANCO OFICIAL.svg (logo en blanco para título)
- /distinto/index.html (landing "Distinto")
- /epk/index.html (EPK / press kit)
- robots.txt
- sitemap.xml

Subida rápida (Netlify):
1. Comprime todos los archivos y carpetas en un ZIP o deja la carpeta descomprimida.
2. Ve a https://app.netlify.com/drop (inicia sesión o crea cuenta).
3. Arrastra la carpeta (o ZIP). Netlify desplegará y te dará una URL pública.

Subida rápida (GitHub Pages):
1. Crea un repositorio en GitHub.
2. Haz upload de todos los archivos (botón "Add file" -> "Upload files") poniendo el contenido en la raíz.
3. En Settings -> Pages activa la rama main y la carpeta / (root) como fuente.
4. GitHub publicará en https://<tu-usuario>.github.io/<repo>

Reemplazos recomendados:
- Si tienes enlaces directos a la pista "Distinto" en Spotify, Apple Music, YouTube, SoundCloud, reemplaza las URLs en /distinto/index.html (hoy están como búsquedas).
- Si quieres la versión del logo exacta (SVG original), reemplaza FALODI LOGO BLANCO OFICIAL.svg por tu SVG original (con el mismo nombre).

Notas:
- Usamos Impact para títulos y Trebuchet MS para cuerpo (sistema), como pediste.
- Favicon es red-mark.svg (círculo rojo). Asegúrate de que el archivo esté en la misma carpeta que index.html.
- JSON-LD y meta tags ya incluidos para indexación legítima.

Si quieres que te envíe esto en un ZIP listo (yo puedo intentar adjuntarlo si tu interfaz lo permite), dímelo y lo preparo; si prefieres, te guío paso a paso para subirlo.
